4) Attachment / Mentalization / Developmental
รายชื่อนักคิดคนสำคัญและผลงานหลัก
หมายเหตุเบื้องต้น
Attachment / Mentalization / Developmental ไม่ใช่ “สำนักเดียว” ในความหมายแคบแบบ Jungian, Freudian, Adlerian หรือ Lacanian
แต่เป็นกลุ่มแนวคิดที่เชื่อมกันผ่านคำถามใหญ่เรื่อง
ความสัมพันธ์แรกเริ่มของมนุษย์
ความผูกพันระหว่างเด็กกับผู้ดูแล
การก่อตัวของตัวตน
การควบคุมอารมณ์
พัฒนาการของจิตใจ
ความสามารถในการเข้าใจสภาวะจิตของตนเองและผู้อื่น
ผลของความสัมพันธ์ต่อบุคลิกภาพและจิตพยาธิวิทยา
ถ้าจะจัดเป็นหมวดในเว็บไซต์ สามารถใช้ชื่อรวมว่า

VI. ข้อสับสนที่ควรระวัง
1) Attachment ไม่ใช่ Object Relations แต่เกี่ยวกันมาก
Attachment Theory มาจาก Bowlby และมีฐานจาก ethology, developmental psychology และ psychoanalysis
ส่วน Object Relations มาจากสาย psychoanalysis เช่น Klein, Fairbairn, Winnicott, Bion
จุดร่วมคือสนใจ “ความสัมพันธ์แรกเริ่ม”
จุดต่างคือ Attachment เน้นระบบความผูกพันและการวิจัยเชิงพัฒนาการมากกว่า
2) Mentalization ไม่ใช่ Attachment โดยตรง
Mentalization เกิดจากการเชื่อม Attachment + Psychoanalysis + Developmental Psychology
Fonagy ใช้ attachment เป็นฐานสำคัญ แต่ประเด็นหลักคือความสามารถในการเข้าใจ “จิตของตนเองและผู้อื่น”
3) Developmental Psychology กว้างกว่า Attachment มาก
Piaget, Vygotsky, Erikson, Bronfenbrenner ไม่ใช่ Attachment theorists
แต่เป็นนักคิดด้านพัฒนาการมนุษย์ที่จำเป็นต่อการทำหมวด Developmental
4) ถ้าเว็บไซต์เน้นจิตบำบัด ควรดันกลุ่มนี้เป็นแกนหลัก
สำหรับเว็บด้านจิตวิทยาเชิงลึกและจิตบำบัด รายชื่อที่ควรเด่นเป็นพิเศษคือ
