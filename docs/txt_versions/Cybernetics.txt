System
ชุดขององค์ประกอบที่สัมพันธ์กันจนเกิดรูปแบบหรือพฤติกรรมบางอย่าง ทั้งระบบชีวภาพ จิตใจ สังคม เทคโนโลยี หรือเว็บความรู้
Boundary
ขอบเขตของระบบ สิ่งที่ถูกนับว่าอยู่ในระบบและอยู่นอกระบบ การกำหนด boundary เป็นการตัดสินใจเชิงความรู้และอำนาจ ไม่ใช่เรื่องกลาง ๆ เสมอไป
Input / Output
สิ่งที่ระบบรับเข้ามาและส่งออกไป แต่ในระบบซับซ้อน input และ output อาจย้อนกลับเป็น feedback ต่อกัน
Feedback
การที่ผลลัพธ์ของระบบย้อนกลับมากระทบพฤติกรรมของระบบเอง
Negative Feedback
feedback ที่ช่วยรักษาสมดุล ลดความเบี่ยงเบน และทำให้ระบบเสถียร
Positive Feedback
feedback ที่ขยายการเปลี่ยนแปลง ทำให้ระบบเร่งตัว เติบโต หรือพังเร็วขึ้นได้
Homeostasis
ความสามารถของระบบในการรักษาสมดุลภายใน แม้สิ่งแวดล้อมเปลี่ยนแปลง
Self-Regulation
ความสามารถของระบบในการปรับตนเองผ่าน feedback และข้อมูล
Control
การกำกับทิศทางของระบบ แต่ใน cybernetics control ไม่ใช่แค่การสั่งจากบนลงล่าง อาจเป็นวงจร feedback ที่กระจายอยู่ในระบบ
Communication
การแลกเปลี่ยนสัญญาณ ข้อมูล ความหมาย หรือความแตกต่างภายในระบบ
Information
สิ่งที่ทำให้ระบบลดความไม่แน่นอนหรือเปลี่ยนพฤติกรรมได้ ไม่ใช่แค่ข้อมูลดิบ
Variety
จำนวนรูปแบบที่ระบบสามารถมีหรือรับมือได้ ยิ่งสิ่งแวดล้อมซับซ้อน ระบบยิ่งต้องมี variety พอ
Law of Requisite Variety
หลักของ Ashby ที่บอกว่าตัวควบคุมต้องมีความหลากหลายพอที่จะรับมือกับความหลากหลายของระบบที่ถูกควบคุม
Open System
ระบบที่แลกเปลี่ยนพลังงาน ข้อมูล หรือวัตถุกับสิ่งแวดล้อม
Closed System
ระบบที่ถูกคิดว่าปิดจากสิ่งแวดล้อมในบางระดับ แต่ใน systems theory ต้องระวังว่าไม่มีระบบชีวิตใดปิดสนิทจริง ๆ
Operational Closure
ระบบอาจปิดในแง่การปฏิบัติการของตนเอง แต่ยังสัมพันธ์กับสิ่งแวดล้อมผ่าน structural coupling
Autopoiesis
ระบบที่ผลิตและรักษาตนเองอย่างต่อเนื่อง เช่น สิ่งมีชีวิตหรือระบบสังคมบางแบบ
Structural Coupling
ความสัมพันธ์ระยะยาวระหว่างระบบกับสิ่งแวดล้อมที่ทำให้ทั้งสองเปลี่ยนแปลงร่วมกัน
Emergence
คุณสมบัติหรือ pattern ที่เกิดจากความสัมพันธ์ของส่วนย่อย แต่ไม่สามารถลดเหลือส่วนย่อยใดส่วนหนึ่งได้ง่าย ๆ
Self-Organization
การเกิด order หรือ pattern จากปฏิสัมพันธ์ภายในระบบ โดยไม่ต้องมีผู้ควบคุมจากศูนย์กลางทั้งหมด
Complexity
สภาวะที่ระบบมีองค์ประกอบจำนวนมาก ความสัมพันธ์หลายระดับ feedback loops nonlinear behavior และผลลัพธ์ที่คาดเดายาก
Complex Adaptive System
ระบบที่ประกอบด้วย agents หลายตัวที่ปรับตัว เรียนรู้ และทำให้เกิด pattern ระดับระบบ
Nonlinearity
ความสัมพันธ์ที่ผลลัพธ์ไม่ได้แปรตามเหตุแบบตรง ๆ เล็กน้อยอาจกระทบใหญ่ หรือกระทบใหญ่บางครั้งกลับได้ผลเล็ก
Delay
เวลาหน่วงระหว่างเหตุและผล ซึ่งทำให้มนุษย์เข้าใจระบบผิดและแก้ปัญหาผิดจังหวะ
Leverage Point
จุดแทรกแซงในระบบที่เปลี่ยนผลลัพธ์ได้มากหรือน้อยต่างกัน จุดลึกที่สุดมักเกี่ยวกับ goal และ paradigm ของระบบ
Attractor
รูปแบบหรือสภาวะที่ระบบมีแนวโน้มเคลื่อนเข้าไปหา
Bifurcation
จุดเปลี่ยนที่ระบบอาจแยกไปสู่เส้นทางใหม่เมื่อเงื่อนไขเปลี่ยนถึงระดับหนึ่ง
Dissipative Structure
โครงสร้างที่เกิดและดำรงอยู่ได้ในระบบเปิดที่แลกเปลี่ยนพลังงานกับสิ่งแวดล้อม โดยเฉพาะในสภาวะไกลจากสมดุล
Second-Order Cybernetics
cybernetics ที่รวมผู้สังเกตเข้าไปในระบบที่ถูกสังเกต
Observer
ผู้สังเกตที่ไม่ได้อยู่นอกระบบอย่างบริสุทธิ์ แต่มีส่วนในการเลือก boundary, language, method และ interpretation ของระบบ
Recursive Loop
วงจรที่ผลของกระบวนการย้อนกลับไปเป็นเงื่อนไขของกระบวนการเอง
Systemic Design
การออกแบบที่มองปัญหาเป็นระบบสัมพันธ์ ไม่ใช่แก้เฉพาะ interface หรือชิ้นส่วนเดี่ยว

จุดเชื่อมกับ Jung-Psychology
Cybernetics / Systems Theory / Complexity Philosophy เชื่อมกับ Jung-Psychology ได้ลึกมาก เพราะ Jung เองไม่ได้มอง psyche เป็นเส้นตรงแบบง่าย ๆ
Psyche ใน Jung เป็นระบบหลายชั้น
มี ego
มี unconscious
มี complex
มี archetypal image
มี compensation
มี dream
มี symbol
มี individuation
มีความสัมพันธ์ระหว่างส่วนย่อยกับทั้งหมด
จุดแรกคือ Compensation ในฐานะ Feedback
Jung มองว่าความฝันและ unconscious มักทำงานเชิง compensatory คือชดเชยหรือถ่วงดุลท่าทีด้านเดียวของ conscious mind
นี่อ่านเป็น feedback loop ได้อย่างสวยมาก
ถ้า ego แข็งไปทางหนึ่ง
unconscious อาจส่งภาพอีกทางหนึ่งขึ้นมา
ไม่ใช่เพื่อทำลาย ego
แต่เพื่อคืนสมดุลให้ระบบ psyche
จุดที่สองคือ Complex ในฐานะ Subsystem
complex ไม่ใช่แค่ “ปม” เฉย ๆ
แต่มันทำงานเหมือน subsystem ที่มี affect, memory, image และ response pattern ของตนเอง
เมื่อ complex ถูก constellate มันอาจเข้ายึด control บางส่วนของระบบ
ego จึงไม่ได้เป็นผู้ควบคุมทั้งหมดเสมอไป
จุดที่สามคือ Shadow ในฐานะ System Exclusion
shadow คือสิ่งที่ ego-system ตัดออกจากภาพตัวเอง
แต่สิ่งที่ถูกตัดออกไม่ได้หายไป
มันกลับมาผ่าน symptom, projection, dream, affect และ relationship feedback
ระบบใดก็ตามที่นิยามตนเองแคบเกินไป จะผลิต shadow มากขึ้น
จุดที่สี่คือ Persona กับ Social Feedback
persona ไม่ได้เกิดจากตัวคนคนเดียว
มันเกิดจาก feedback ระหว่าง ego กับสังคม
คนอื่นคาดหวังอะไร
ระบบให้รางวัลกับบทบาทไหน
สังคมลงโทษอะไร
เราถูกอ่านแบบไหน
เราปรับตัวอย่างไรเพื่ออยู่รอด
Persona จึงเป็น cybernetic interface ระหว่าง psyche กับ social system
จุดที่ห้าคือ Individuation ในฐานะ Self-Organization
individuation ไม่ใช่แผนสำเร็จรูปที่ ego สั่งให้เกิด
แต่มันคล้ายกระบวนการ self-organization ของ psyche ทั้งระบบ
เมื่อ ego, shadow, anima/animus, complex, dream, symbol และ Self เริ่มสัมพันธ์กันใหม่
pattern ใหม่ของตัวตนจึงค่อย ๆ เกิดขึ้น
จุดที่หกคือ Self ในฐานะ Organizing Principle
Self อาจถูกอ่านเป็น organizing principle ของ psyche
ไม่ใช่ตัวตนเล็ก ๆ แบบ ego แต่เป็นศูนย์กลาง/ขอบเขตเชิงระบบที่ทำให้ psyche ทั้งหมดมีแนวโน้มไปสู่ wholeness
จุดที่เจ็ดคือ Dream ในฐานะ System Signal
ความฝันไม่ใช่ภาพสุ่มไร้ค่า
แต่อาจเป็น signal จากส่วนอื่นของระบบ psyche
มันบอกว่า conscious attitude กำลังพลาดอะไร
ระบบต้องการชดเชยอะไร
และมี pattern อะไรกำลังเกิดขึ้นใต้ระดับคำพูด
จุดที่แปดคือ Projection ในฐานะ Feedback Error
projection คือการอ่านข้อมูลจากโลกผ่านภาพภายในของเราเอง
ในเชิงระบบ นี่คือ feedback ที่ถูกบิดโดย model ภายใน
เราคิดว่าเราตอบสนองต่อคนจริง
แต่บางครั้งเราตอบสนองต่อ imago หรือ complex ที่เราวางทับเขา
จุดที่เก้าคือ Cultural Complex
Systems Theory ช่วยให้เข้าใจว่า complex ไม่ได้อยู่แค่ในบุคคล
แต่มีระดับครอบครัว ชุมชน วัฒนธรรม และสังคม
ชาติ ศาสนา เพศ race class trauma และประวัติศาสตร์สามารถกลายเป็น feedback system ที่สร้าง affect และ response pattern ซ้ำ ๆ ในคนจำนวนมาก

เว็บนี้เองสามารถถูกมองเป็นระบบภายนอกของ psyche ได้
